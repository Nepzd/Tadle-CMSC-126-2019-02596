This program asks the user to input the following information of a Filipino citizen:
1. Annual Income
2. 13th month pay
3. Total Annual mandatory deductions (e.g. SSS, GSIS, Pag-ibig, Philhealth)
4. Number of dependents (e.g. Old parents, Children below 21 yrs old)

Then, the program computes the following:

1. Gross income = Annual Income + 13th month pay
2. Total exemptions for dependents. Take note that if there are 5 dependents, then only 4 can avail the P50T exemptions per dependent.
        • Total exemptions = 50000 * n dependents, if n <=4 • Total exemptions=200000, if dependents> 4
3. Total deductions = mandatory exemptions + personal exemptions + total exemptions for dependents. Personal exemptions per tax payer = 250,000.
4. Taxable Income = Gross income – Total deductions
5. Payable Tax
